Rules

Add server: it only takes keys in its arc — from the previous server up to itself, (A, D]. Those keys used to belong to B.

Remove server: D's keys slide to the next server clockwise (B). All other keys stay put.

Simplified demo: 24 sample keys on a 0–360° ring. Compare with modulo hashing where ~75% of keys move when going from 3→4 nodes.
